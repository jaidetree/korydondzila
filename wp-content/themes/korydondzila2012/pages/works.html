Works

These are Maya 3D MEL scripts used to create custom & practical effects within Maya. These are completely written from scratch.

  • kdAutoRig

    A prototype for an auto rigger.

  • kdBarycentrics

    A prototype for a barycentric rigging system.

  • kdCityGen

    A random city generator. Shown in demo reel.

  • kdGrass

    A prototype for a grass generator and simulator. Shown in demo reel.

  • kdScriptTool

    A prototype for a scripting interface. Originally done as a joke, but it became an exercise in UI scripting.

  • kdSegmentJoint v1.1

    A rigging tool that splits up a joint into equal parts.
